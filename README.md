# AI3000 Clean Hover Landing Page

這是移除 Hero 右側浮動圖片卡後的 GitHub Pages 一頁式 HTML。

## 本版調整
- Hero 改為影片/動畫背景層 + 上方文字內容。
- 移除右側圓角浮動 DM 卡。
- 保留滑鼠 hover 動態：卡片浮起、圖片放大、按鈕光掃、導覽底線、展示區光影。
- 可直接放到 GitHub Pages。

## 使用方式
上傳全部檔案到 GitHub repository，Settings → Pages → Deploy from branch。

## 替換 Hero 影片
目前 Hero 使用 `assets/hero-style-reference.png` 作為背景。若有 mp4，可自行把 hero-media 改成 video 標籤。
